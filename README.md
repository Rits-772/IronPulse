# IronPulse | Elite Fitness Protocol

IronPulse is a high-performance, cyberpunk-themed fitness tracking platform designed for athletes who treat their training like engineering. Merging raw iron with deep neural analytics.

![IronPulse Screenshot](apps/frontend/public/icon-512.png)

## 🚀 Neural Capabilities
- **3D Command Center**: Immersive landing page with real-time scroll depth optics.
- **Matrix Nutrition**: Interactive Calorie Orbit and Macro breakdown bars.
- **The Oracle Analytics**: Predictive 1RM modeling and Muscle group saturation heatmaps.
- **Neural Upgrades**: Gamification system with XP and Rank (Level) progress.
- **Haptic Rest Timer**: Optimized training density with precise countdown protocols.
- **Visual Identity**: Mobile-optimized UI with Avatar integration.

## 🛠 Tech Stack
- **Frontend**: React, Vite, TailwindCSS, Framer Motion
- **3D Graphics**: Three.js, React Three Fiber (R3F)
- **Backend/Auth**: Supabase (PostgreSQL, Storage, Auth)
- **Charts**: Recharts
- **PWA**: Manifest & Mobile optimization for standalone protocol access.

## 📡 Deployment (Vercel Ready)
This project is configured for seamless deployment on Vercel.
1. **Root Directory**: `apps/frontend`
2. **Build Command**: `pnpm build`
3. **Output Directory**: `dist`
4. **Env Variables**: Add `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
